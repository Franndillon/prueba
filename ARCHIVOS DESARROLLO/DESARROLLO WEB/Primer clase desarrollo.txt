Cuando ingresamos esa dirección en un navegador, esta se traduce en un IP a través de una base de datos distribuida conocida como DNS. 
Este IP corresponde a un ordenador lejano llamado Servidor.
Ejemplo Google: 172.217.0.131
Este Servidor atiende la solicitud y responde con los contenidos de la página Web deseada.
El ordenador recibe estos contenidos y “dibuja” la página web.
