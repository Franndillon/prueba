Elementos Semánticos

Toda etiqueta en HTML tiene una conotación semántica. Gráficamente, y se ven igual pero cuando los motores de búsqueda leen la etiqueta saben que el los elementos que contiene son relevantes para la página y, por lo tanto, quedan destacados en la misma. Por otra parte, existen etiquetas HTML que sólo tienen conotación semántica y, por lo tanto, no afectarán a la visualización pero que son fundamentales para el armado y posicionamiento de una página web.

Elementos que no pueden faltar:

  • Header: Referente al título de la página
  • Nav: Navegación de la página
  • Main: Contenido principal de la página
  • Footer: Información adicional (copyright)

Otros elementos semánticos:

Todos se ubican dentro del main

  • Secction: Para generar una sección dentro de una página
  • Article: Para agregar artículos
  • Aside: Barra lateral
  • Figure: Permite vincular semánticamente una imagen con su descripción (figcaption)

Vínculos de navegación

Los vínculos (o links) nos permiten conectar nuestra web con otras o distintos elementos en nuestra propia web. Existen dos tipos de vínculos:

  • Relativos: Utilizados generalmente para interconectar elementos de nuestra propia wbe
  • Absolutos: Utilizados generalmento para traer elementos de webs externas
  • Anclas: Las anclas nos permiten crear una navegación entre distintas secciones de una misma página. Para armar una navegación entre secciones, hay que crear anclas (elementos a con el atributo id) y un link a esa ancla (elementos a con atributo href)

Vínculos de funcionalidad

Existen vínculos que nos permiten desatar una función distinta a la de navegación, como por ejemplo generar una descarga o mandar un mail. Cada función corresponde a un atributo de la etiqueta a diferente.

  • Mandar mail: Un vínculo de mail se crea agregando el atributo mailto:"direcciondecorreo"

A la hora de elegir una tipografía para nuestra página podemos utilizar las fuentes seguras o generar la importación de una fuente externa. Google Fonts es una página propiedad de google que contiene fuentes de uso libre que podemos importar a nuestra web para aplicarla utilizando la propiedad font-family

Pseudo-Clases

Las Pseudo-Clases son un tipo de selector CSS que nos permiten modificar las propiedades de los elementos en determinados o cuando se cumplan ciertas condiciones. Un elemento en el que se vé claramente esto es el elemento a:

  • a:hover = Cuando le pasamos el mouse por arriba
  • a:link = Cuando todavía no visitamos el link
  • a:visited = Cuando ya visitamos el link
  • a:active = Cuando el link se encuentra activo
